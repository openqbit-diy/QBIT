# QBIT

 mini QBit (mini quantum computer - diy "do it yourself").

Release 2021

HARDWARE & SOFTWARE (Q-BLOCKLY)

The OpenQbit processor has 6 gates:


*Pauli-X Gate

*Pauli-Y Gate

*Pauli-Z Gate

*Hadamard Gate

*User defined X

*Y-Rotation Gates


With Q-Superposition & Q-Entanglement



Q-API integration of commercial quantum computers.
